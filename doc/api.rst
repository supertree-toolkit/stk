STK API
=======

The STK has been split such that the main functionality can be imported into
your own Python script and used programmatically. If you have downloaded the
source package there are a number of example scripts in the stk/scripts
directory. If you know Python, simply import the stk module with:

.. code-block:: python
    
    import stk

and all the functions below will be available as stk.XXXX.
